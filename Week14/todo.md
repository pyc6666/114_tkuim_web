# 課程管理系統 UI 設計規劃

## 📋 整體佈局結構

### 1. 頂部導航欄 (Top Navigation Bar)
- **背景色**: 深藍灰色 (#3a4a5c)
- **高度**: 60px
- **內容**:
  - 左側: Logo + 系統名稱
  - 中間: 主要導航連結（選課、課程、成績等）
  - 右側: 用戶資訊 + 通知圖標 + 登出按鈕

### 2. 側邊欄 (Sidebar)
- **寬度**: 250px
- **背景色**: 淺灰色 (#f5f7fa)
- **內容區塊**:
  - **用戶資訊卡片**
    - 圓形頭像 (80px)
    - 用戶名稱
    - 學號/角色
  - **我的課程** (導航菜單)
    - 🏠 首頁
    - 📚 我的課程 (高亮顯示，使用青綠色 #00bcd4)
    - 📢 公告
  - **我的課表** (次級菜單)
    - 📋 備審
    - ✅ 互動教材
    - 👤 個人題庫
    - 📊 Rubrics
  - **校外競賽** (次級菜單)
    - 👥 個人/認證

### 3. 主內容區 (Main Content Area)
- **背景色**: 白色 (#ffffff)
- **內邊距**: 30px

---

## 🎨 主要功能區塊設計

### A. 搜尋與篩選區 (Search & Filter Section)

#### 搜尋框
- **位置**: 主內容區頂部
- **樣式**:
  - 寬度: 100%
  - 高度: 50px
  - 邊框: 1px solid #e0e0e0
  - 圓角: 8px
  - 佔位符文字: "加入課程"
  - 右側按鈕: "搜尋" (青綠色背景 #00bcd4)
- **功能**: 
  - 提示文字: "搜尋: 最近課程"
  - 支援課程名稱、課程代碼搜尋

#### 篩選器組 (Filter Group)
- **佈局**: 3列 × 2行的網格佈局
- **每個下拉選單**:
  - 寬度: 自適應 (flex-grow)
  - 高度: 40px
  - 邊框: 1px solid #ddd
  - 圓角: 4px
  - 背景: 白色
  
- **篩選項目**:
  1. **學年**: 下拉選單 (預設: 全部)
  2. **學期**: 下拉選單 (預設: 全部)
  3. **課程性質**: 下拉選單 (預設: 通行中)
  4. **開課單位**: 下拉選單 (預設: 全部)
  5. **年級**: 下拉選單 (預設: 全部)
  6. **進度**: 下拉選單 (預設: 全部)

---

### B. 課程卡片列表 (Course Card List)

#### 單個課程卡片設計
- **佈局**: 水平排列
- **尺寸**: 
  - 寬度: 100%
  - 高度: 120px
  - 間距: 15px (卡片之間)
- **樣式**:
  - 背景: 白色
  - 邊框: 1px solid #e8e8e8
  - 圓角: 8px
  - 陰影: 0 2px 8px rgba(0,0,0,0.08)
  - Hover 效果: 陰影加深 + 輕微上移

#### 卡片內容結構 (從左到右)

**1. 課程縮圖區 (100px × 100px)**
- 漸層背景 (淺藍到淺紫)
- 課程相關圖標/插圖
- 圓角: 8px
- 左側邊距: 10px

**2. 課程資訊區 (主要內容)**
- **課程標題**
  - 字體大小: 18px
  - 字重: 600 (Semi-bold)
  - 顏色: #333
  - 可點擊 (hover 變色)

- **課程詳細資訊** (小字體，灰色 #666)
  - 第一行: `課程代碼: 1141TLMX051M300-0B`
  - 第二行: `開課系(所): 1 B 甲班 2025-04-01 ~ 課程結束日期: 2026-01-31`
  - 第三行: 圖標組
    - 📁 總資料夾: 必修
    - 📊 學分數: 1.0
    - 🎯 完課判定: 👤👤 (圖標表示)

**3. 操作按鈕區 (右側)**
- **課程概述** 按鈕
  - 樣式: 文字按鈕
  - 顏色: #00bcd4
  - Hover: 底線效果
  
- **更多** 按鈕
  - 樣式: 文字按鈕
  - 顏色: #00bcd4
  - Hover: 底線效果

- **課程編號標籤**
  - 位置: 右上角
  - 背景: 淺灰色
  - 文字: "1141"
  - 字體大小: 12px

## 🎯 互動設計細節

### Hover 效果
- **課程卡片**: 
  - 陰影: 0 4px 16px rgba(0,0,0,0.12)
  - 位移: translateY(-2px)
  - 過渡: 0.3s ease

- **按鈕**:
  - 顏色加深
  - 底線動畫

### 點擊效果
- **課程標題**: 進入課程詳情頁
- **課程概述**: 顯示課程簡介彈窗
- **更多**: 展開操作選單

## 🎨 色彩系統

### 主色調
- **主要色**: #00bcd4 (青綠色) - 用於按鈕、高亮

- **背景色**: #f5f7fa (淺灰) - 用於側邊欄

- **次要文字**: #666666

## 📱 響應式設計

### 桌面版 (> 1200px)
- 側邊欄固定顯示
- 課程卡片單列顯示
- 篩選器 3列2行佈局

### 平板版 (768px - 1200px)
- 側邊欄可收合
- 課程卡片單列顯示
- 篩選器 2列3行佈局

### 手機版 (< 768px)
- 側邊欄隱藏，使用漢堡選單
- 課程卡片垂直堆疊
- 篩選器單列顯示，可滾動

## ✨ 動畫效果

### 頁面載入
- 課程卡片: 從下往上淡入 (stagger 效果)
- 延遲: 每個卡片間隔 50ms

### 篩選切換
- 課程列表: 淡出 → 更新 → 淡入
- 持續時間: 300ms

### 側邊欄導航
- 選中項目: 背景色滑動動畫
- 圖標: 輕微縮放效果

## 🔧 技術實現建議

### HTML 結構
```
- header.top-nav
- div.container
  - aside.sidebar
    - div.user-card
    - nav.main-menu
    - nav.sub-menu
  - main.content-area
    - section.search-filter
      - input.search-box
      - div.filter-grid
    - section.course-list
      - article.course-card (重複)
```

### CSS 框架
- 使用 Flexbox 進行佈局
- CSS Grid 用於篩選器網格
- CSS Variables 管理色彩系統
- 使用 transition 實現平滑動畫

### JavaScript 功能
- 篩選器聯動邏輯
- 課程卡片動態載入
- 搜尋即時過濾
- 側邊欄收合控制

## 💡 設計亮點

1. **清晰的資訊層級**: 使用字體大小、顏色、間距區分資訊重要性
2. **高效的篩選系統**: 多維度篩選，快速定位目標課程
3. **視覺化課程資訊**: 使用圖標和標籤，降低閱讀負擔
4. **流暢的互動體驗**: 豐富的 hover 和過渡效果
5. **一致的設計語言**: 統一的色彩、圓角、間距系統
